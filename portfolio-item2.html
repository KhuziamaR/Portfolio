About The Color Game

This project was built using HTML5, CSS3 and Javascript and was inspired by Colt Steeles Web Development BootCamp! This project simply generates random numbers, ranging from 0-255 to represent a rgba() color. The goal of this project is that it allows users to improve there front end design and styling skills by allowing them to visualize how colors look in rgb values! This project helped to improve my front end development skills as it helped me learn more about DOM manipulation as well as how to use HTML, CSS and Javascript together to make full front end applications.

Challenges

One particular challenge I encountered while making this website was not getting the exact functionality that I was trying to achieve at first, however, to solve this issue, I simplified the code by refactoring repeated lines into functions which allowed me to keep track of exactly what is going on. Functionality was the most important piece to this project, however, I will be working on the design and styling to further improve it!
