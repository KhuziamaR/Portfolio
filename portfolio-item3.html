About The To Do List

This project was built using HTML5, CSS3 and Javascript and was inspired by Colt Steeles Web Development BootCamp! The goal of this project is to help users stay more organized! The user can insert tasks they wish to complete by entering a new task in the "Add New Todo" textbox, cross out todos by clicking on the Todo they wish to crossout, or even permanently delete the todo by clicking the trashcan icon! This project helped to improve my front end development skills as I learned how to use HTML, CSS and Javascript together to make full front end applications.
